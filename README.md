# Traffic Planner Dashboard

React-pohjainen kuljetussuunnittelun dashboard-prototyyppi.

## Ominaisuudet

- Kuljetusten KPI-näkymä
- Kuljetusten tilamerkinnät
- Kapasiteettivaroitukset
- Kuljettajan ajoaikavaroitukset
- EU ajo- ja lepoaikariskin tarkistus
- Suodatus: kaikki / myöhässä / korkea prioriteetti
- Haku kuljetustunnuksella

## Teknologiat

- React
- Vite
- JavaScript
- CSS
- Git / GitHub

## Projektin tarkoitus

Tämä projekti osoittaa käytännössä:

- logistiikan prosessiosaamista
- kuljetussuunnittelun työnkulkujen ymmärrystä
- dashboard-ajattelua
- frontend-kehitystä Reactilla

## Live Demo

https://Vode72.github.io/traffic-planner-dashboard

## English summary

A React-based logistics planning dashboard prototype for monitoring transports, capacity, driver hours and EU driving time risk.
